## 📂 Estructura del Proyecto

Para facilitar la gestión y el despliegue, el proyecto se divide en dos módulos independientes. Por favor, selecciona la carpeta correspondiente para ver su guía de instalación:

### 🎨 [Frontend (React)](./front-react)
> **Directorio:** `front-react`  
> Contiene la interfaz de usuario, gestión de estados y consumo de servicios API.  
> [👉 Ver instrucciones del Frontend](./front-react/README.md)

### ⚙️ [Backend (Flask)](./server-flask)
> **Directorio:** `server-flask`  
> Contiene la lógica de negocio, procesamiento de datos y endpoints REST en Python.  
> [👉 Ver instrucciones del Backend](./server-flask/README.md)
